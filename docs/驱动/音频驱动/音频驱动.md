## 音频输出配置

```shell

运行 alsamixer -c 1。
按 F6 确保选中的是 rockchipes8388。
不要只看音量条，请按左右方向键遍历所有控件。
重点寻找以下类型的控件，并确保它们处于 <OO> (开启) 状态，而不是 <MM> (静音) 或 Off：
Power / Enable: 如 DAC Power, ADC Power, Codec Power, Headphone Power。
Mux / Route: 如 Headphone Mux, Speaker Mux, Output Mux, Left DAC Mux。
如果是 Mux 选项，按 M 键或上下键切换，尝试切换到 DAC, Headphone, 或 Mixer 等选项，直到看到波形或听到声音。
Switches: 如 Headphone Switch, Speaker Switch, Playback Switch。
操作技巧：
遇到 <MM> 按 M 键消除静音。
遇到 Off 的开关，按上下键改为 On。
遇到 Mux 选择器，轮流切换不同的值测试。
特别注意 ES8323: 有时需要开启 Headphone Jack Detect 相关的旁路，或者手动强制开启输出而不依赖插入检测。


```
