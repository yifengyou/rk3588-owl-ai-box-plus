# 风扇驱动

* 板子没有独立风扇控制模块，通过GPIO控制输出的高低电平控制启停

## 风扇接口控制方法：


方法一：执行如下三行命令，第三行的1，0对应风扇开关。

```shell

echo 153 > /sys/class/gpio/export
echo out > /sys/class/gpio/gpio153/direction
echo 1 > /sys/class/gpio/gpio153/value
````

---

方法二：修改dts编译内核后烧录kernel目录下zboot.img

dts中添加：如下usb供电配置

```shell
vbuswind_pwr: vbuswind-pwr {
    compatible = "regulator-fixed";
    regulator-name = "vbus5v_pwr";
    gpio = <&gpio4 RK_PD1 GPIO_ACTIVE_HIGH>;
    enable-active-high;
    regulator-always-on;
    regulator-boot-on;
    regulator-min-microvolt = <3300000>;
    regulator-max-microvolt = <3300000>;
};
````
